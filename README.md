Project Overview
This project presents an interactive Customer Shopping Behavior Dashboard designed to analyze purchasing patterns, customer demographics, and revenue distribution. The dashboard enables stakeholders to understand customer preferences and optimize business strategies for increased sales and customer satisfaction.

Objective
->Analyze customer purchasing behavior
->Identify top-performing product categories
->Understand customer demographics (age, gender, subscription status)
->Track revenue and sales trends
->Support data-driven decision-making for marketing and sales strategies

Tools & Technologies:
Data Visualization: Power BI / Excel Dashboard (update if needed)
Data Processing: Microsoft Excel
Data Cleaning: Power Query / Excel
Version Control: Git & GitHub

The dataset includes the following attributes:
Customer ID
Gender
Age Group (Young, Middle Age, Old)
Subscription Status
Product Category (Clothing, Accessories, Footwear, Outerwear)
Purchase Amount
Review Rating
Shipping Type

Key Insights:
Majority of customers are non-subscribers (~74%)
Clothing category generates the highest revenue and sales
Accessories contribute significantly as a secondary category
Balanced purchasing behavior across age groups
Opportunities exist to convert non-subscribers into subscribers
